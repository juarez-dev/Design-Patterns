# Design-Patterns // not final readme

This repository contains my practice and learning notes about different **Design Patterns** in Java.

The goal of this project is to understand how each pattern works and implement practical examples for every one of them.

## What you will find in this repository

- Explanations of different Design Patterns
- Simple and clear Java implementations
- Practical examples of when and why to use each pattern

## Design Patterns included

### Creational Patterns
- Singleton
- Factory Method
- Abstract Factory
- Builder
- Prototype

### Structural Patterns
- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

### Behavioral Patterns
- Observer
- Strategy
- Command
- Iterator
- Mediator
- Memento
- State
- Template Method
- Visitor

## Purpose

This repository is mainly for **learning and practice**, as well as to serve as part of my **developer portfolio**.

Each pattern will include:
- Explanation
- UML or structure
- Java example
- Use case
